# Speechful - Landing page per un’app di lingue (Vue.js + Tailwind + DaisyUI)

Speechful è una landing page responsive per un’app fittizia dedicata all’apprendimento delle lingue.  
Il progetto nasce come esercizio front-end per lavorare su layout moderni, micro-interazioni e componenti riutilizzabili usando **Vue.js**, **Tailwind CSS** e **DaisyUI**.

## About

La pagina simula il sito di presentazione di un prodotto SaaS: un app di nome Speechful che ha come obiettivo l'apprendimento semplificato, da parte del potenziale utente, delle lingue estere attraverso delle micro-lezioni, degl esercizi interattivi e focus sulla pronuncia. 

## Obiettivi del progetto

- Realizzare una **landing page moderna** per un prodotto fittizio, curando tipografia, spaziature e gerarchia visiva.
- Esercitarmi con **Vue.js** e la creazione di componenti riutilizzabili (es. `StatCounter`).
- Gestire micro-interazioni come:
  - animazione dei numeri solo quando diventano visibili,
  - sticky layout con contenuto che resta ancorato durante lo scroll.
- Simulare un caso reale di front-end per SaaS, con sezioni classiche: hero, about, features, stats, why us, call to action, footer.

## Linguaggi utilizzati

- **Vue.js/Vite**
- **Tailwind CSS** (per lo stile)
- **DaisyUI** (libreria di componenti)
- **Animazioni / UX:**
  - Transizioni con classi Tailwind (`transition`, `opacity`, `translate`, ecc.),
  - Counter animato con `requestAnimationFrame` e `IntersectionObserver`.

## Note

Speechful è un progetto puramente dimostrativo: non esiste un backend reale e tutti i contenuti (testi, numeri, feature) sono pensati come placeholder per esercitazione sul framework front-end "Vue.js".
